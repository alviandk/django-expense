Documentation of django-expense
===============================

This documentation covers the django-expense, a simple personal
expense tracker application for `Django
<http://www.djangoproject.com>`_.

For sapmle screenshots or detailed desription see :ref:`userguide`.

The source code of the project is hosted on `bitbucket.org/szunyog/django-expense <https://bitbucket.org/szunyog/django-expense/>`_ and the latest
version of this documentation can be found on `django-expense.readthedocs.org <http://django-expense.readthedocs.org/>`_.
